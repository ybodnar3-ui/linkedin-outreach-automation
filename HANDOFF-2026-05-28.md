3. **Web Search implementation** (`extension/background.js`):
   - `searchWebForLinkedInProfiles()` ‚Äî uses Google ‚Üí Bing ‚Üí DDG fallback
   - Opens search engine in active tab, scrapes results, closes
   - 4 parsing strategies: anchors, cite tags, body regex, script JSON
   - Decodes Bing `bing.com/ck/a?u=a1<base64>` and Google `/url?q=` redirects

4. **Batch Web Search** (`extension/popup.html` + `popup.js` + `background.js`):
   - UI: keywords textarea √ó locations textarea
   - Backend: `runBatchWebSearch()` iterates Cartesian product
   - Pushes leads to backend every 3 queries (won't lose progress on crash)
   - Stop button works
